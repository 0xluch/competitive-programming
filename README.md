# Competitive Programming

Solutions to problems from UVa Online Judge, Codeforces, and *Programming Challenges* (Skiena), written in C++ as I work through them.

## Structure
- `uva/` — UVa Online Judge solutions
- `codeforces/div3/` — Codeforces Div. 3 solutions
- `skiena-challenges/` — Problems from Skiena's *Programming Challenges*
- `utils/` — reusable templates/snippets
